⚙️ Installation & Setup
1️⃣ Clone Repo
git clone https://github.com/thesayancodes/decentralised_social_media.git
cd decentralised_social_media
2️⃣ Install Dependencies
npm install
3️⃣ Run Blockchain
ganache-cli
4️⃣ Deploy Contracts
truffle migrate --reset
5️⃣ Start App
npm start
